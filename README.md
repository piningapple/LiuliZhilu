Для работы приложения необходима база словаря в формате html, ежедневную выгрузку базы можно скачать на сайте БКРС(https://bkrs.info/p47, внизу страницы «Словарные базы за сегодня», их там 3) и преобразовать с помощью PHP конвертера(https://github.com/dictz/php-multiconverter):
    • Поместите файлы и каталоги мультиконвертера в чистый корневой публичный каталог локального сервера.
    • В каталог da/ распакуйте 3 словарные базы
    • Запустите локальный сервер, откройте интернет-проводник и перейдите через адресную строку по адресу локального сервера.
    • В открывшейся странице мультиконвертера укажите нужные пункты
    • Конвертация может длиться долго
Готовую html базу поместите в html_dabkrs_bruks/dabkrs.html
Запустите файл database_dict.py, он преобразует html файл в базу данных(sqlite)
